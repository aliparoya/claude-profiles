# Intro

This is the landing page for the docs site. Replace it with whatever the project actually is. A one-paragraph What. A short list of pages a new reader should hit first. A link to the [style guide](./style-guide/) so future contributors know how to write.

The site is set up so any MDX page can drop in MUI components without extra wiring. The theme is in `src/theme/mui/`; the `<ThemeProvider>` is already wrapped around every page in `src/theme/Root.tsx`.

## Where to start

- [Style guide](./style-guide/): how to write for this site.
- [MDX reference](./style-guide/mdx-reference): copy-paste snippets for slide decks, diagrams, math, admonitions, and responsive images.
- [Image prompts](./style-guide/image-prompts): prompt patterns for generated hero imagery.

## Setup

```bash
pnpm install
pnpm start
```

Dev server runs on `http://localhost:3000`. Hot reload picks up edits to `docs/`, `blog/`, and `src/`.
